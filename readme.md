> - 测试前，请先用 real 目录内的程序替换 cgi-bin 目录下的程序
> - 测试后，请用 simulation 目录内的程序替换 cgi-bin 目录下的程序

### 天气预报

- 根据IP地址获取主机位置，主动获取天气
- 可以查询国内任意城市的天气，国外天气经测试暂不支持
- 根据天气内容，自动替换对应的天气图标

### 科技新闻

- 获取科技类新闻，可以通过 `阅读原文` 访问原网页
- 链接国内知名科技资讯网站
